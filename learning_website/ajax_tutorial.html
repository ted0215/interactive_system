Ajaxの基本的な実装方法

Ajaxは基本的にこれらの技術を用いて作っていきます。

  • XMLHttpRequest
  • JavaScript
  • DOM
  • XML

Ajaxはこのように色々な技術を組み合わせて実装しています。

それぞれの技術を簡単に説明します。

 

XMLHttpRequest

クライアントとサーバーの間でデータを伝送するための機能をクライアント側で提供する API です。

簡単にまとめると、ブラウザ上でサーバーと HTTP 通信を行うための API です。

この API を使って実装を行なっていくのが JavScript です。

JavaScript

XMLHttpRequestはjavascriptの組み込みオブジェクトです

したがって、JavaScriptを使わないと実装ができません。

DOM

ここでは、HTMLや、XMLを「ツリー構造」として展開して、加工や変更をしやすくするためのものです。

XML

文書やデータの意味や構造を記述するためのマークアップ言語の一つ（HTMLと似たようなもの)です。

XMLはタグを自由に設定でき、そのタグに意味づけをすることができます。

「XML」を使えば、複数のデータを同時にやりとりしても、どのデータがどの要素なのか一発で判明でき流のが強みです。
